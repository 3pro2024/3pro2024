# 手話ぷら - 手話学習アプリケーション

手話を楽しく学べるWeb学習アプリケーション

## プロジェクト概要

「手話ぷら」は、手話を学びたい人のためのインタラクティブな学習プラットフォームです。クイズ形式で手話を学習し、レベル別・ランク別に体系的に手話を習得できます。YouTube動画と連携し、実際の手話動作を視覚的に確認しながら学習を進めることができます。

### 主な機能

- **クイズモード**: レベル別（初級・中級・上級・方言）とランク別（2級～5級）で手話を学習
- **学習モード**: 手話データの一覧表示とキーワード検索機能
- **実績システム**: 学習進捗の可視化と達成度の記録
- **動画連携**: YouTube埋め込みによる手話動作の視覚的学習
- **レスポンシブデザイン**: PC・タブレット・スマートフォンに対応

## 開発環境・使用言語

### 技術スタック

- **フレームワーク**: Vite 6.3.5
- **言語**: TypeScript 5.8.3
- **スタイル**: Vanilla CSS（相対単位ベースのレスポンシブデザイン）
- **データ形式**: JSON
- **ストレージ**: LocalStorage（学習進捗の永続化）
- **動画配信**: YouTube Embed API
- **パッケージマネージャー**: pnpm

### 開発ツール

- ESLint: コード品質管理
- Prettier: コード整形
- Commitizen: コミットメッセージの標準化

## セットアップ方法

### 前提条件

- Node.js 18以上
- pnpm（または npm/yarn）

### インストール手順

```bash
# リポジトリのクローン
git clone https://github.com/3pro2024/3pro2024.git
cd 3pro2024

# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev
```

ブラウザで `http://localhost:5173` にアクセス

### ビルドと本番環境

```bash
# 本番用ビルド
pnpm build

# ビルド結果のプレビュー
pnpm preview
```

## プロジェクト構造

```
3pro2024/
├── src/

│   │   ├── index.html   # タイトル画面

│   │   ├── learn/       # 学習モード

│   ├── types/          # TypeScript型定義
│   └── utils/          # ユーティリティ関数

```

## 使用方法

1. **タイトル画面**: 「クイズモード」または「学習モード」を選択
2. **モード選択**: レベル・ランク・キーワードで学習内容を絞り込み
3. **クイズ**: 手話動画を見て正しい意味を選択
4. **学習**: 手話データを自由に閲覧・検索
5. **実績確認**: 学習進捗と達成度を確認

## 工夫したところ

- **Multi-Page Application (MPA) 構成**: Viteを活用し、各ページを独立させることで高速なページ遷移と保守性を実現
- **相対単位ベースのCSS設計**: px使用を避け、rem/em/vw/vh/clamp()を活用したアクセシブルなレスポンシブデザイン
- **TypeScript型安全性**: 厳格な型定義により、バグを開発時に検出
- **コンポーネント設計**: 関数型コンポーネントパターンによる再利用性の高い実装
- **テーマシステム**: CSS Custom Propertiesを使った柔軟なテーマ切り替え機能
- **LocalStorage活用**: ユーザーの学習進捗を永続化し、継続的な学習体験を提供
